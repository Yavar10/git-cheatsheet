Git Cheat Sheet

Essential Git commands for developers

1. Getting Started

git config

Configure Git with your user information

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

git init

Initialize a new Git repository

git init

git clone

Clone a remote repository to your local machine

git clone https://github.com/username/repository.git

2. Creating a Repo & First Push

git remote add

Add a remote repository

git remote add origin https://github.com/username/repository.git

Initial Commit & Push

Add, commit, and push your first changes

git add .
git commit -m "Initial commit"
git push -u origin main

3. Everyday Workflow

git status

Check the status of your working directory

git status

git add

Add files to the staging area

git add filename.txt
git add .
git add -A

git commit

Commit staged changes with a message

git commit -m "Your commit message"
git commit -am "Add and commit in one step"

git rm

Remove files from Git and working directory

git rm filename.txt
git rm --cached filename.txt

4. Working with Branches

git branch

List, create, or delete branches

git branch
git branch new-feature
git branch -d branch-name

git checkout

Switch between branches or restore files

git checkout branch-name
git checkout -b new-branch
git checkout -- filename.txt

git merge

Merge one branch into another

git merge feature-branch
git merge --no-ff feature-branch

5. Collaborating with Remotes

git fetch

Download changes from remote without merging

git fetch origin
git fetch --all

git pull

Fetch and merge changes from remote

git pull origin main
git pull --rebase origin main

git push

Upload your commits to remote repository

git push origin main
git push -u origin feature-branch

6. Inspecting the History

git log

View commit history

git log
git log --oneline
git log --graph --oneline

git diff

Show differences between commits, branches, or files

git diff
git diff --staged
git diff commit1 commit2

git show

Show information about a specific commit

git show
git show commit-hash
git show HEAD~1

7. Undoing Mistakes

git reset

Reset your repository to a previous state

git reset HEAD~1
git reset --hard HEAD~1
git reset --soft HEAD~1

git revert

Create a new commit that undoes changes

git revert commit-hash
git revert HEAD

git stash

Temporarily save changes without committing

git stash
git stash pop
git stash list
git stash apply

git clean

Remove untracked files and directories

git clean -n
git clean -f
git clean -fd
